What is the difference between HTML elements and tags?
An HTML element is the combination of the HTML tags (opening and closing) and the content inside them. It is everything from the start of the opening tag to the end of the closing tag, including the content.
Tags are the markup used to define HTML elements. They are the actual parts of HTML code that are used to structure and display content.

What is semantic HTML?
Answer Semantic HTML refers to the practice of writing HTML code in a way that clearly describes the meaning (or "semantics") of the content within the webpage. In other words, it involves using the appropriate HTML elements to represent the structure of the page, making it more readable for both humans and machines (such as search engines or screen readers)

What are the entities in HTML?
Answer In HTML, entities are used to represent special characters that have a reserved meaning in HTML syntax or characters that may not be easily typed using a standard keyboard. These entities are represented by an ampersand (&), followed by a name or a number, and ending with a semicolon (;).

Why is a URL encoded in HTML?
Answer URLs are encoded in HTML to ensure that special characters like spaces, ampersands (&), question marks (?), etc., are correctly transmitted and interpreted by web browsers and servers. Encoding replaces these characters with a percent-encoded format (e.g., a space becomes %20).

Does a < ! DOCTYPE html> tag is a HTML tag?
Answer No, the < ! DOCTYPE html> declaration is not an HTML tag. It is a declaration that informs the browser about the version of HTML to use (HTML5 in this case).

What is difference between HTML and XHTML?
Answer HTML is a lenient markup language, while XHTML is a stricter version of HTML, following XML rules (e.g., all tags must be closed, attributes must be quoted).

What are logical and physical tags in HTML?
